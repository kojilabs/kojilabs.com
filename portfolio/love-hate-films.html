Love Hate Films

Technologies Ruby on Rails, JavaScript, HTML5, CSS3
Devices Desktop, Tablet, Mobile
Project Length 3 Months

Love Hate Films approached us to build a movie reviews website that featured gamification to keep users engaged.

Using Ruby on Rails, we designed and built a site that uses points-based game mechanics to insentivise users to review films. As users post more reviews they earn points, allowing them to unlock additional 'levels' that showcase their reputation to other user.

We decided to implement social sign-up with Facebook and Google+ to make it really simple for new users to get started on the site.

In addition to a regular desktop website, we really wanted to make Love Hate Films work well on phones and tablets. Following best practices of responsive web design, we engineered a design that would gracefully adapt to the screen size of the device the site is being viewed on.

This website was built using the latest web standards, notably HTML5 and CSS3.

In addition to the website, we were also responsible for revamping the Love Hate Films branding. We designed a new logo that would accomodate the company's desire to expand into new verticals in the future.
