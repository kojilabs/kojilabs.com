Seaton Gifts

Technologies Tictail, JavaScript, HTML5, CSS3
Devices Desktop, Tablet, Mobile
Project Length 2 Months

Seaton Gifts approached us to create them a web store to complement their existing bricks and mortar outlets.

The client wanted an admin interface that would be easy to use. It was clear early in the project that Tictail would be the best software solution to power the site.

The Tictail interface is simple and intuitive. Updating product listings is easy, and managing customer orders has never been more straightforward.

We designed a bespoke Tictail theme for the store and handled all the initial set up.

The final design is clean and minimalistic. Incorporating the company's strong brand with a bold header section and logo.

We also introduced a premium web font that's in-keeping with the Seaton Gifts brand.

The layout collapses gracefully to optimise the browsing experience on smaller screens.

We built this project using HTML5, CSS3, and JavaScript.
